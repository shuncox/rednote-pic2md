# 小红书图片转Markdown工具 - 用户使用手册

## 简介

小红书图片转Markdown工具是一个简单易用的桌面应用程序，可以帮助您将小红书上的多页图片文章转换为Markdown格式的文档。本工具支持多种OCR服务，能够自动识别系列文件，并生成格式化的Markdown文档。

## 系统要求

- 操作系统：Windows 10+、macOS 10.14+、Linux
- Python 3.8+
- 网络连接（用于OCR服务）

## 安装和启动

### 1. 环境准备

确保您的系统已安装Python 3.8或更高版本。

### 2. 下载和安装

1. 下载项目代码到本地
2. 打开终端或命令提示符，进入项目目录
3. 使用uv安装依赖：
   ```bash
   uv sync
   ```

### 3. 启动应用程序

```bash
uv run python run.py
```

## 使用方法

1. **选择图片文件**: 点击"选择图片文件"按钮，选择一张小红书图片
2. **配置OCR服务**: 在设置面板中配置您的OCR服务API密钥（配置会自动保存）
3. **设置输出选项**: 选择输出目录和文件名
4. **开始转换**: 点击"开始转换"按钮
5. **预览和保存**: 在右侧预览区域查看结果，确认无误后保存

## OCR服务申请指南

### 百度OCR

1. 访问[百度智能云控制台](https://console.bce.baidu.com/)
2. 注册并登录账号
3. 创建"文字识别"应用
4. 获取API Key和Secret Key

### 腾讯云OCR

1. 访问[腾讯云控制台](https://console.cloud.tencent.com/)
2. 注册并登录账号
3. 开通"文字识别"服务
4. 获取Secret Id和Secret Key

### 阿里云OCR

1. 访问[阿里云控制台](https://ecs.console.aliyun.com/)
2. 注册并登录账号
3. 开通"文字识别"服务
4. 获取Access Key ID和Access Key Secret

## 文件名格式要求

本工具支持小红书标准文件名格式：
```
{标题}_{页码}_{作者}_来自小红书网页版.{扩展名}
```

示例：
```
一个很笨但能成功写出SSCI的小tips_2_博士小导师_来自小红书网页版.jpg
```

支持的图片格式：
- PNG
- JPG/JPEG
- BMP
- GIF
- WEBP

## 常见问题

### Q: 为什么无法识别某些图片？
A: 可能的原因：
- 图片质量过低
- 图片中的文字过于模糊
- OCR服务配置错误
- 网络连接问题

### Q: 如何提高OCR识别准确率？
A: 建议：
- 使用高分辨率的图片
- 确保图片中的文字清晰
- 选择合适的OCR服务
- 检查API配置是否正确

### Q: 转换后的文档格式不正确怎么办？
A: 可以：
- 手动编辑生成的Markdown文档
- 检查原始图片质量
- 尝试使用不同的OCR服务

### Q: 程序启动失败怎么办？
A: 请检查：
- Python版本是否为3.8+
- 是否正确安装了所有依赖
- 操作系统是否支持

### Q: 如何批量处理多个系列？
A: 建议：
- 每次处理一个系列
- 确保文件名格式正确
- 检查文件是否完整
