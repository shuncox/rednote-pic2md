# 小红书图片OCR转成markdown格式文档

## 项目描述

小红书上的很多文章是以多页图片形式存在的，本项目的任务就是将这些图片转换成markdown格式的文档。

## 项目要求

- 使用python编程，uv管理环境
- 代码格式规范，有适当的注释
- 使用Qt作为UI界面框架
- 可接受常用图片格式

## 业务流程

1. 用户打开文件选择，选中一个图片文件

2. 程序自动生成需要转换的文件列表。文件名形式如下：

```
一个很笨但能成功写出SSCI的小tips_2_博士小导师_来自小红书网页版.jpg

...

一个很笨但能成功写出SSCI的小tips_9_博士小导师_来自小红书网页版.jpg
```

用户只需选中其中一个文件，其它文件名可根据文件名中的序号推测出来。

3. 对文件列表进行排序，注意是按上述序号的数字大小排序
4. 用户按下“转换”按钮，程序依次对图片文件进行OCR，形成一个markdown文档
5. 对文档进行格式检查，要注意页面之间的段落衔接

## 其它

- 目标文件可以输出至源文件所在目录，或者由用户制定目录，并且能够保存状态
- 文件列表可以由用户进行添加、删除、排序、拖拽调整文件位置等
- OCR直接使用成熟的API服务，用户可以设置API Key等
- 在docs目录下输出本项目的具体设计文档
- 测试文档在test目录下
